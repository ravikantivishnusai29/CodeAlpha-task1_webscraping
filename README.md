# 📚 Task 1 - Web Scraping

## 📌 Project Overview

This project was completed as part of my **Data Analytics Internship at CodeAlpha**.

The objective was to scrape book information from the **Books to Scrape** website using Python and store the extracted data in a structured CSV file for further analysis.

---

## 🎯 Objective

- Extract book information from a website.
- Store the scraped data in CSV format.
- Prepare the dataset for future data analysis tasks.

---

## 🌐 Website

https://books.toscrape.com/

---

## 🛠️ Technologies Used

- Python
- Requests
- BeautifulSoup
- Pandas
- Jupyter Notebook

---

## 📂 Dataset Information

The dataset contains **100 books** with the following attributes:

- Title
- Price
- Rating

---

## 📁 Output

- **books_100_records.csv**

- Data Collection
- Data Cleaning
- CSV File Creation
